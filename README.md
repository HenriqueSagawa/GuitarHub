# 🎸 Music School Manager

## Sobre o projeto

O **Music School Manager** é um sistema web desenvolvido para auxiliar na gestão de aulas de violão, permitindo o gerenciamento de turmas, alunos, presenças, avisos e materiais didáticos, além de oferecer um portal do aluno com acesso a conteúdos e ferramentas úteis para estudo.

O objetivo do sistema é centralizar a organização das aulas em uma única plataforma, facilitando o trabalho dos professores/coordenadores e melhorando a experiência dos alunos.

---

## Funcionalidades (MVP)

O sistema contará com dois tipos de usuários: **Administrador** (professor/coordenador) e **Aluno**.

### Administrador

* Cadastrar alunos
* Criar e gerenciar turmas
* Definir horários das aulas
* Registrar presenças e faltas
* Enviar avisos
* Enviar materiais didáticos
* Enviar cifras
* Acompanhar frequência dos alunos

### Aluno (Portal do Aluno)

* Visualizar horários das aulas
* Consultar presenças e faltas
* Visualizar avisos
* Acessar materiais
* Visualizar cifras
* Utilizar afinador de violão online
